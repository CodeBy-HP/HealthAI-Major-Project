# Sample Lab Reports for Testing

## Where to Get Test Reports

### Option 1: Use Sample Images Online
Search for "sample cardiac lab report" or "sample blood test results" on Google Images. Download any medical report image for testing.

### Option 2: Create Your Own Test Report

Create a simple text file or image with these common cardiovascular metrics:

```

Patient: John Doe
Date: 2024-01-15

BLOOD PRESSURE
Systolic: 120 mmHg
Diastolic: 80 mmHg

LIPID PANEL
Total Cholesterol: 180 mg/dL
LDL Cholesterol: 100 mg/dL
```

### Option 3: Use Any File

The system works with mock data, so you can upload ANY image or PDF to see the analysis in action with realistic sample results.

## Understanding Test Results

### Blood Pressure (Normal: 120/80 mmHg)
- **Normal**: < 120/80
- **Elevated**: 120-129/<80
- **High**: ≥130/≥80

### Cholesterol (Normal: <200 mg/dL)
- **Desirable**: <200
- **Borderline High**: 200-239
- **High**: ≥240

### Blood Sugar (Normal: 70-100 mg/dL)
- **Normal**: <100
- **Prediabetes**: 100-125
- **Diabetes**: ≥126

### Heart Rate (Normal: 60-100 bpm)
- **Low**: <60
- **Normal**: 60-100
- **High**: >100

### BMI (Normal: 18.5-24.9)
- **Underweight**: <18.5
- **Normal**: 18.5-24.9
- **Overweight**: 25-29.9
- **Obese**: ≥30

## Testing Tips

1. **Start Simple**: Upload any image file to see the demo in action
2. **Real Reports**: Use actual lab reports (redact personal info) for realistic AI analysis
3. **Multiple Tests**: Try different report types to see AI adaptation
4. **Compare Results**: Upload same report twice to see consistency

## Privacy Note

- No data is stored on servers
- All processing is temporary
- Files are analyzed and immediately discarded
- Use anonymized reports for privacy
